DevFlow - Agile Kanban Board

A modern, lightweight Kanban board built to mirror real-world Agile workflows and CI/CD practices. This project was specifically built to align with Kiprosh's engineering principles: TDD, Automation, Microservices, and Developer Happiness.

ReactNode.jsMongoDBJest

 Key Features

Kanban Board: Drag and drop tasks across "To Do", "In Progress", and "Done" columns (Embracing Agile/Scrum methodologies).

Automated Sanity Testing: 100% Backend API tested using Jest and Supertest following TDD practices.

Microservices Structure: Separated Authentication/Task logic concept and structured for scalable microservices.

Developer Happiness: Clean, intuitive UI with instant feedback and dark mode ready.

🛠️ Tech Stack
Frontend: React.js, Axios, @hello-pangea/dnd (Drag and Drop)
Backend: Node.js, Express.js
Database: MongoDB Atlas (Cloud)
Testing: Jest, Supertest

Version Control: Git & GitHub
